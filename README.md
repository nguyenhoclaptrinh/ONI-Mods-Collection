# Oxygen Not Included - Mods Collection

A curated collection of essential mods for Oxygen Not Included, modernized and repaired for the latest game versions (Build 679336+).

## Featured Mods

### 1. Customizable Speed
An updated version of the classic speed control mod.
- **Modernized Core**: Rebuilt using the latest SDK-style project format for better compatibility.
- **PLib Integration**: Updated to the latest PLib (v4.x) to ensure stability with the current game engine.
- **Fixed Logic**: Resolved issues where speed settings wouldn't persist or caused jitters during emergency events.

### 2. TiengViet (Vietnamese Translation)
A comprehensive repair of the popular Vietnamese translation mod.
- **Syntax Overhaul**: Fixed over 1,000 broken link tags and mismatched style markers that caused "Strip Link Formatting" failures in the game logs.
- **Semantic Logic Fixes**: Corrected critical mismatches where logic signal colors (Red/Green) were swapped in item descriptions.
- **Parameter Patching**: Restored missing placeholders (`{0}`, `{Hotkey}`) to prevent runtime errors and display correct game data.

## Installation

1. Download the mod folders from this repository.
2. Navigate to your ONI local mods directory:
   `Documents\Klei\OxygenNotIncluded\mods\Local\`
3. Copy the `CustomizableSpeed` and `TiengViet` folders into the `Local` directory.
4. Enable the mods in the game's main menu.
